STEP 2: COMMON FIXES
===========================================

Fix 1: MongoDB Extension Issue (Most Common!)

Problem: MongoDB extension Render free tier pe nahi hai

Solution:
1. composer.json file open karo
2. MongoDB dependency remove karo:

FROM:
  "mongodb/mongodb": "^1.17"

TO:
  (Remove this line completely)

3. Git push karo:
   git add composer.json
   git commit -m "Remove MongoDB for Render"
   git push

4. Render pe redeploy karo
